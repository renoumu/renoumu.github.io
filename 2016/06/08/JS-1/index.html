JavaScript -- 笔记

发表于 2016-06-08   |   分类于 JavaScript

###JAVASCRIPT组成

  • ######ECMAScript: 解释器、翻译 – 几乎没有兼容性问题
  • ######DOM:Document Object Model (文本对象模型) – 有一些兼容性问题
  • ######BOM:Browser Object Model (浏览器对象模型) – 几乎不兼容

###变量的类型（1）

类型 typeof()运算符：用于检测变量的类型；
–’string’,’number’,’boolean’,’object’,’function’,’undefined’,
出现undefined有两种可能:
    1.没有定义;
    2.定义了,但是没给值
一个变量应该只存放一种类型的数据

###变量的类型（2）

parseInt()从字符串中提取数字;(转成整数)
parserfloat();(提取小数)
isNaN()可以判断一个字符是不是NaN;

###隐式类型转换

#####”==”等于 and “===”全等

#####”+”加号,”-“减号,”*”乘号,”/“除号
加号：可用于字符串和数字相加
减号：只用于数字相减
乘号：只用于数字相乘
除号：只用于数字相除

###变量作用域

局部变量：定义在一个函数里，只能在这个函数里面用;
全局变量：不定义在任何函数里面，可以在任何地方用，尽量不要用

###命名规范

1.匈牙利命名法
类型前缀 :

首字母大写 :
例如：oDiv;

驼峰命名法 :
例如：UserName;
