storage事件：

  • [x] 当同源的localStorage或sessionStorage有更改以后，会触发这个事件。
Property Type Description
key string 被增加、修改、删除的数据的key
oldValue Any 数据被修改之前的value、如果数据是被增加的则为null
newValue Any 数据被修改之后的value
url string 修改storage页面的url

sessionStorage:

  • [x] sessionStorage为临时性保存数据，当页面关闭就会消失。其他一切与localStorage一样
  • [x] sessionStorage不能跨页面访问，也不会触发跨标签页的storage事件。它只局限在当前的标签页里。
