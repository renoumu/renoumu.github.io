发表于 2016-05-20   |   分类于 JavaScript

JavaScript的引入方式：

内部书写：

在html文件中直接进行代码的书写,建议写在body下面;

  • <script type="text/javascript">这里书写JavaScript的代码</script>

外部引入：

在内部引入的基础上加上”src=xxxx.js”,可以放在放在页面的元素中,
也可以把JavaScript引用放在中.

  • <script type="text/javascript" src="javascript.js">这里书写JavaScript的代码</script>
